# Yle Morgonkollen

## Motivation

I wanted to get the TLDR of the main news stories that [Svenska Yle](https://svenska.yle.fi) makes on weekdays delivered to my mail.
I could not find such an option on the main site and decided to make my own.

### Installation

- Install the dependencies listed in requiremennts.txt
- Supply your own [Sendgrid](https://sendgrid.com) API key.
- Specify the sender and recipient mail addresses in the script.
